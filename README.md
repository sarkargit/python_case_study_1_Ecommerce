# Customer Purchase Behavior Analysis

## Overview

This project involves performing a detailed data analysis on a dataset containing information about customer purchases. The dataset includes various attributes such as customer information, purchase prices, payment methods, language preferences, and more. The goal is to derive insights that can be used for understanding customer behavior, product trends, and spending patterns.

### Objective
The primary objectives of this project are:
- To perform exploratory data analysis (EDA) on a customer purchase dataset.
- To answer a series of predefined questions to uncover useful insights about customer behavior.
- To explore key aspects such as purchase prices, credit card usage, preferred languages, and other important customer attributes.

---

## Dataset Description

The dataset consists of multiple columns, including:
- **Email**: Customer email address.
- **Purchase Price**: Price of the product purchased.
- **Job Title**: Job title of the customer.
- **Language**: Preferred language of the customer.
- **IP Address**: The IP address from which the purchase was made.
- **Credit Card Provider**: The credit card provider used for the purchase.
- **Credit Card Expiry Date**: Expiry date of the customer's credit card.
- **Purchase Time**: Timestamp of the purchase (AM/PM).
- **Purchase Location**: Location where the purchase was made.
